# HTTP

- 웹에서 브라우저와 서버 간에 데이터를 주고받기 위한 방식
- 상태가 없는 프로토콜
- TCP/IP 통신 위에 동작
- 기본 포트는 80번

Hypertext -> Hypermedia (JSON, mp3, mp4..)
request - response

HTTPS - Hypertext Transfer Protocol Secure

- SSL / TLS 암호화된방법으로 데이터를 주고 받음

100 : information
200 : success
300 : redirection
400 : client error
500 : server error

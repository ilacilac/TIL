**npm** : node package manager 
패키지를 관리한다.
개발환경에 필요한 라이브러리 하나하나가 패키지

**npx** : 일회성으로 npm을 불러온다.

> npm으로 불러오면 node_modules에 쫙- 저장됨 

```
npx create-react-app my-app 
```

개발환경(webpack)을 직접 셋팅해보는게 사실좋음.



라우팅으로 보여주는 페이지를 선택할 수 있다.

> Ex) 이 path가 이루어질 때(우리가 정하는것), 이 컴포넌트를 보여줄거야
